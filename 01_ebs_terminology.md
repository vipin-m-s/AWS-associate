# EBS terminology IOPS, baseline IOPS

## IOPS
input output operation per second  a storage can handle

## Baseline IOPS
This is the minimum guaranteed IOPS provided by the storage

## provisioned iops
We can also set how much iops we want from the storage

## burst performance / burst credit
- When we do not utilize the IOPS
- The remaningin IOPS is stored as credit
- It will be utilized during burst IOPS\
- Temporary performance boost above the baseline iops

## Throughput 
How much data can be transffered per second
