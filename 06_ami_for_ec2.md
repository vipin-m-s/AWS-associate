# AMI 

- Amazon machine image is image for operating system. Which consists of operating system + packages.
- We can launch ec2 instance from the AMI
- We can install additional packages + software in the ec2 instance
- We can then create custom AMI from that ec2 instance.
- AMI are regional service, We cannot use AMI of mumbai region in hyderabad region.
- We can copy the AMI from mumbai to hyderabad and use it there.
- We can sell AMI in aws marketplace if customer wants to use our product.
