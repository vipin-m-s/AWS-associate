# launch template

- When we create an ec2 instancce, there are lot of settings which are available
- It is difficult to select these settings everytime we create an ec2 instance
- In this case, we can create a launch template with all the parameters we want
- Then we can launch an ec2 instance with the launch template
- We can enforce best practices via the launch template,
- We can create multiple version of launch template with ddifferent set of parameeters
- If some values are not selected part of the launch template, We can define it when creating an ec2 instance
