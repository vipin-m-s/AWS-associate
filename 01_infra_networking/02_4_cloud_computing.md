# cloud computing 

Cloud computing is On demand delivery of IT resources over the internet in pay as you go model. 
Example:- VM, storage

# Cloud computing models

Software as a service - software is provided over the internet thorugh a browser example google doc, google sheet, netflix
Platform as a service - Platform to develop and run the software is provided exampl Elastic bean stalk 
Infrastructure as  a service - also known as hardware as a service, using compute, storage over the internet. Example ec2, s3. 

# Multi cloud 

Organizations using different cloud providers to achieve its needs is called multi cloud 

# Characteristics of cloud computing

## on demand and self service
Inorder to provision resources, We should be able to provision at any time by our own without asking the provider.

## Elasticity 
Suppose a company wants to 
- deploy 2 GB ram server
- Scale the server to 8 GB ram for 1 day
- Scale down the server back to 2 GB

The user must be able to scale up or scale down the resources easily

## MEasured service
Suppose a company wants to run the server only for 30 minutes. 
He should not be charged for entire month. 
He should be charged only for 30 minutes. 
He should be charged only for the duration he has used the service.
