# One zone IA

By default in all the storage classes
THe data is replicated to minimum 3 availabilizty zones
SO even if the AZ goes down the data is accesible
Hence the cost is also more for storing and replicating data 

## With the one zone io 
The data is stored only in 1 AZ
If the AZ goes down then the data stored there is lost. WHich is rare
The cost of this type of storage is cheaper.
