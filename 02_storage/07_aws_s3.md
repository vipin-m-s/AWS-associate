# s3
Suppose we use external hard disk to store data
If we have only 1 hard disk then if it get corrupted, we will loose all our data
So we can have additional hard disk which we can replicate. 
Both the hard disk should not be present in same location 
If there is a natural calamity we will loose all the data
Hence we can make use of cloud storage to store all our data, 
The cloud provider is responsible for high availability and durabbility of our data.

AWS s3 is a storage service
We create bucket and objects. 
bucket is container for storing folder and files.
The files we upload to the s3 bucket is called object
