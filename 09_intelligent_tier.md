# Intelligent tier

S3 standard -> costly 
S3 infrequent access -> cheaper
When objects in standard storage is not accessed for 30 consequitive days. 
The objects in the standard storage is moved to the infrequent access storage
There will be continous monitoring of objects and it will be moved
If the obejct in infrequent access tier is accessed. 
It is moved back to the standard tier.
It will help organizations to save costs.
