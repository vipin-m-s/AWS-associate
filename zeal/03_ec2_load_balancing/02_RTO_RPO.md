# RTO 
recovery time objective - how long will it take to make the infrastructure up and running
How long will it take to migrate the data to other region 
Ex- If RTO is 3 hours, DR region should always be ready for switch over

# RPO 
recovery point objective - How far back in time data have we lost
How far back in time do we have the data 
How fr back in time data loss can we tolerate
Ec:- If RPO is 5 hours, We need to take backup of 5 hours.
