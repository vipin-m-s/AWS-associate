# AWS budgets

- We can set budget , which will alert us when the actual or forecasted spend crosses the budget
- Provides great detail in which service costing more money

There are multiple templates to set budgets
- Zero budget plan :- when cost is 0.01 > free tier limits
- monthly budget plan :- We can set custom budget for the month. Example 10$ etc.

There are 3 email alerts by default 
- When the forecasted spend is expected to > 100% 
-  When The actual spend is > 85%
- When the actual spend is > 100%
