# RDS backup 
- rds supports automatic, manual and scheduled backups
- We can restore to any point in time using automated backups to a specific time.
- Automatic backups happen in a daily window chosen by customer or by AWS
- They are retained for 35 days
- Manual backups can be retained to any day chosen by customer
- Snapshots can be copied to any region or aws account.
- IT backs up the database and transaction logss as well
- backups are called snapshots
