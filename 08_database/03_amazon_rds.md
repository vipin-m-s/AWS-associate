- It is fully managed relation database service offered by AWS
- It offers multiple database options
  - amazon - aurora
  - open source - mysql, postgres, mariadb
  - commercial - oracle db, ibm db2
- AWS hanldes the infrastructure, os/database upgrade, backup, patching, security, backup etc

# Deploy
1) create ec2 instance
2) Create mysql rds instance
3) install mysql client in ec2 instance
4) connect to rds
