# RDS  vs Aurora

## databse engine
- RDS supports myql, postgres, mariadb, oracle, ibm db2
- aaurora supports mysql, postgres, dsql

## Replicas
- 15 replicase ( 5 for oracle ) 
- 15 replicas

## compute and storage
- RDS couple storage and compute
- Aurora - decoupled storage and compute

## performance
- rds low compared to aurora
- aurora high compreed to rds

## cost
- rds - low compared to aurora
- aurora - high compared to rds

## storage 
- rds - 64 TB
- Aurora - 128 TB

## Features
- RDS read replica, multi az, snapshot, backup
- Aurora above plus global database, machine learning, cloning, serverless
