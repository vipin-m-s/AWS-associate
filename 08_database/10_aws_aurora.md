# AWS Aurora 
- It is fully managed relational database service built by AWS
- It is compatible with postgres and mysql
- It provides 5x more performance than mysql and 3x more performance than postgres with 1/10th of the cost
- It has seperated the compute and storage. In others the compute and storage is tied.

- The aurora consists of primary instance and read replicase. It supports upto 15 read replicas.
- The aurora maintains about 6 copies of the data across 3 availabilityt zones
