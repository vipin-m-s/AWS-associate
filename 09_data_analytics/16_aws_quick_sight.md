# quick sight
- It is used for generating interactive visual dashboards from data
- It is used for business analytics purposes
- It can be integrated with existing applications
- Pricing is per user session or it can be ued for wieder audience using Enterprise plan
- It supports RDS, aurora, s3 opensearch ,redshift, athena as data source
- We can also directly add csv, json files to quicksight for adhoc query
- Use case:- vizualization and ad hoc queries.
