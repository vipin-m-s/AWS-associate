# AWS redshift
- It is fully managed data warehouse solution from Amazon
- Supports Peta bytes of data
- Supports streaming data, databases with low code/no code - zero ETL
- data can be analyzed by quick sight or third party tools like tableu
- data can be shared to different region, third party data warehouse solutions without data movement/copy
- Supports 2 types of options
  - serverless
  - provisoned cluster
