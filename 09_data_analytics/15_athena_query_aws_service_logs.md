# Query AWS service logs
- We can use athena to query aws service logs such as cloudtrail events, elb access logs or VPC flow logs
- We can use SQL statements for querying.
- AWS services support storing the logs and evenets in S3 bucket
- We can query the logs stored in s3 bucket
