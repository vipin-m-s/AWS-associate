# EMR
- If we want to make use of third party ETL such as hadoop, spark, hive and presto we can use EMR
- It is a full managed service for third party frameworks
- It supports using ec2, eks and serverless for compute
- If we use ec2 instance we can use autoscaling and spot instances for saving cost. 
- For storage HDFS ( ebs/instance store volume ) If using EMRFS (s3) used as storage
