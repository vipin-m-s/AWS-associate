## OLTP vs OLAP

### OLTP
- online transaction processing
- Used for applications to store data
- small amount of reads and writes

### OLAP 
- online analytic processing
- Used for storing large amounts of data PBs
- Huge read operations

## data warehouse
- used for store large amounts of data
- expensive to maintain the data warehouses
- Data is take from warehouse and moved to other data base for prcoessing

## data lakes 
- central repositroy for sstoring, processing data
- It is cheaper compared to data warehouses

## data format for data lakes
- row based format :- csv
- column format :- parquet
- table based format :- allows delete/updates, ACID property :- AWS s3 supports this, Apache iceberg
