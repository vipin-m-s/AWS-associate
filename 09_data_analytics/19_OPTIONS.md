# redshift serverless
- Fully managed service with automatic sclaing
- We are not provided access to underlying infrastructure
- Used for unpredictable workloads

# redshift provisioned cluster
- We will be provided with a cluster
- Cluster has a master node and several compute nodes for processing.
- Data node will share the requests to compute nodes for distributed processing
- We can manually scale or schedule the scaling
- Used for predicatble workloads
