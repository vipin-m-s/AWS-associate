# Amazon macie

- Companies use s3 buckets to store petabytes of data 
- They might want to expose the data in the s3 bucket to other aws account or to the public
- There might be chances that PII personally identifiable information will be stored in s3 bucket
- we do not want such information to be shared with other AWS account or public.
- We want to protect using Amazon Macie service
- It is a Full managed service by AWS.
- It can monitor and protect sensitive data in S3.
- It uses machine learning and pattern matching to identify sensitive data at scale in s3
- Trigger alery when it identifies the sensitive data
