# s3
simple storage serice
- one of the earliest service by aws
- It is a web based cloud storage service provided by aws and has virtually unlimited space
- It provide 11 9s of durability which means 1 object in 100 billion object can be lost in a year
- It is used in wide variety of usse cases such as storage for
  - enterprise applications
  - media store
  - data lakes
  - hosting static websites
