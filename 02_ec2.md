# EC2

## What is an ec2 instance ? 
- It is virtual machihne provided by AWS
- AWS manages the physical harcdware and type 1 hypervisor.
- We do not get access to underlying hardware
- We are provided with just the virutal machine.
- We can sleect whcih OS the instancer will have

## Importatn attributes of EC2 instance
- name - tag
- operting system - AMI
- vCPU / memory - Instance type 
- private network - VPC
- firewall - security group
- SSH keypair
- root volume - EBS
- configurations scripts - user data
- EC2 instance connect

## Ways to connect to ec2 instance
- ssh client
- RDP client
- ec2 instance connect
- AWS session manager
